# 残心 — Vector Genesis Engine

実験的リサーチフレームワーク  
〜 単純な相互作用から生まれる創発的構造・補償ダイナミクス・持続性について

## 概要

VGE（Vector Genesis Engine）は、単純な局所ルールから複雑な構造や持続的な振る舞いがどのように生まれるかを探るための実験的システムです。

本プロジェクトの出発点は、以下の問いでした。

> 「持続的な自己組織化を生み出すために、最小限必要なメカニズムとは何か？」

VGEは、あらかじめ与えられた目的関数に向かって最適化するのではなく、以下の要素とその相互作用から生まれるダイナミクスを観察することを目的としています。

- Difference（差異）
- Constraint（制約）
- Compensation（補償）
- Persistence / Residue（持続・残渣）

---

## 核心となる考え方

最も単純なレベルで、個体は「現在の状態」と「何らかの制約」との間に差異を経験します。  
その差異が補償的な応答を生み、結果として残渣（Residue）が残ります。  
そしてその残渣が、未来の応答の方早いそのものを変えていきます。

```
Constraint（制約）
    ↓
Difference（差異）
    ↓
Compensation（補償）
    ↓
Residue（残渣）
    ↓
Future Difference（次の差異）
```

このループは、明示的な目的や計画子が存在しないにもかかわらず、まるで目的を持っているかのような振る舞いを生み出します。

## 探索の歴史

### Void Core（最初の阶段）
最初期の実験は、単純に結合された2つのエージェントで行われました。

A ←→ B

ノイズが存在する中でも、同期パターンが自然に生まれることが観測されました。  
そこで得られた最も重要な気づきは、

> 「関係性そのものが、動的な対象になった」

ということでした。状態ではなく、**差異**が実体として振る舞い始めたのです。

### VGE（拡張）
モデルを多体ベクトルシステムに拡張したところ、新たな現象が現れました。

- Role Persistence（役割の持続）
- CounterGeneratorの自発的生成
- Mode遷移
- 長期的な構造形成

これらはすべて、外部から役割を割り当てたわけではなく、相互作用の中から自然に生まれてきたものです。

### DAP（Delayed Amplification Phenomenon）
長時間シミュレーションの中で、小さな差異が長期間にわたって貯積し、後になって不均衡な影響を与える現象が観測されました。  
これにより、「持続すること自体が、受動的な記憶ではなく能動的な力学成分である可能性」が示喷されました。

### Residue（残心としての解釈）
最も重要な発見の一つが、Residueの振る舞いでした。

当初は単なる「記憶」として扱っていましたが、複数のアブレーション実験を通じて、以下のような解釈が浮上してきました。

> Residueは、単なる過去の記録ではなく、**未来の応答趨向（感受性）そのものを保持する状態変数**である。

イベントが外見上終わった後でも、Residueが残っていることで、次の刺激に対する反応の方早いが変わり続けます。  
これは武道における「残心」に近い力学と言えます。

---

## 主な実験的知見

- **相互作用（Interaction）を切ると**、長期的な増幅現象が大きく弱まる
- **制約（Constraint）を切ると**、CounterGeneratorの自発的生成がほぼ停止する
- **Residue（持続）を切ると**、活動自体は残るものの、長期的なコヒーレンスや構造の安定性が低下する
- どの要素を単独で切っても、創発的な振る舞いが完全に消えることはなかった

これらの結果から、VGEは単一の根源メカニズムによって驅動されているのではなく、**複数のレイヤーが相互に支え合うネットワーク**として機能していると考えられます。

## 現在の作業仮説

現在の最も有力な作業仮説は以下の通りです。

> 「創発的な振る舞いは、未解決の差異が持続（Persistence）と相互作用しながら時間の中で展開されることで生まれる」

制約が差異を生み、差異が補償を驅動し、補償が残渣を残し、残渣が未来の感受性を変える——このループが続続としています。
